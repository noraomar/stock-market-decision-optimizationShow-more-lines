# Stock Market Decision Optimization

## Overview

This project simulates stock market decision-making using a Markov Decision Process (MDP). The model evaluates Buy, Sell, and Hold actions under different market conditions and determines the optimal investment strategy using Value Iteration and Policy Iteration.

## Market States

- Bull Market
- Bear Market
- Neutral Market

## Actions

- Buy
- Sell
- Hold

## Technologies

- Python
- NumPy
- Matplotlib
- Markov Decision Process (MDP)
- Value Iteration
- Policy Iteration

## Features

- Dynamic market transition probabilities
- Reward-based decision optimization
- Optimal policy extraction
- Value function estimation
- Convergence comparison between algorithms

## Results

- Computed optimal policies for different market conditions
- Compared convergence speed of Value Iteration and Policy Iteration
